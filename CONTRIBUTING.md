# Contributing

This repository is a portfolio sample but follows a simple docs workflow.

## Add a new article
1. Create the Markdown file under the correct section in `/docs`
2. Use the template: `/templates/kb-article-template.md`
3. Add the new article to the table of contents in `docs/index.md`

## Quality checklist
- Headings are consistent
- Steps are clear and complete
- Links work (no 404s)
- Terminology follows `STYLE-GUIDE.md`
