# Create an account

**Applies to:** All users  
**Last updated:** 2026-01-11

## Overview
Create a TaskFlow account so you can access your workspace and start managing projects.

## Steps
1. Open the TaskFlow sign-up page.
2. Enter your `Full name`, `Email address`, and `Password`.
3. Select **Create account**.
4. Verify your email address using the verification link sent to your inbox.

## Result
You can sign in to TaskFlow and create or join a workspace.
