# Invite teammates

**Applies to:** Workspace admins, Project owners  
**Last updated:** 2026-01-11

## Overview
Invite teammates so they can collaborate on projects and tasks.

## Steps
1. Go to `Settings > Workspace settings`.
2. Select **Members**.
3. Select **Invite members**.
4. Enter one or more email addresses.
5. Choose a role (for example, Member or Admin).
6. Select **Send invites**.

## Result
Invited users receive an email invite link to join your workspace.
