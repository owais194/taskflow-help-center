# Create your first project

**Applies to:** End users  
**Last updated:** 2026-01-11

## Overview
Projects help you organize tasks by team, initiative, or client.

## Before you begin
- You must be signed in to TaskFlow.

## Steps
1. In the left sidebar, select **Projects**.
2. Select **New project**.
3. Enter a `Project name` (for example, “Website Redesign”).
4. (Optional) Add a short description.
5. Select **Create project**.

## Result
Your project is created and ready for tasks.
