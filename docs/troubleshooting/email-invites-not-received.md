# Fix “email invite not received”

**Applies to:** All users  
**Last updated:** 2026-01-03

## Overview
If you were invited to a workspace but didn’t receive the invite email, use the steps below.

## Steps
1. Check spam/junk folders.
2. Search your inbox for “TaskFlow invite”.
3. Ask the inviter to resend the invite:
   - `Settings > Workspace settings > Members > Pending invites`
4. Confirm the inviter used the correct email address.

## Admin checks (if you're an admin)
- If your organization uses email filtering, allowlist the TaskFlow sender domain.
- If the workspace restricts allowed domains, confirm the invite matches the allowed domain list.
