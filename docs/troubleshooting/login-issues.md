# Troubleshoot login issues

**Applies to:** All users  
**Last updated:** 2026-01-03

## Overview
Use this guide to resolve common login issues, including password problems, SSO issues, and verification email delays.

## Issue: “Invalid email or password”
Try the following:
1. Confirm you're using the correct email address.
2. Reset your password using **Forgot password**.
3. If your workspace uses SSO, try signing in via **Continue with SSO**.

## Issue: Verification email not received
1. Check spam/junk folders.
2. Search your inbox for “TaskFlow verification”.
3. Request a new verification email.

## Issue: 2FA codes not working
1. Ensure your device time is set automatically.
2. Try generating a new code.
3. Use backup codes (if provided) or contact your workspace admin.
