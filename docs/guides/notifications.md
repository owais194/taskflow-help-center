# Manage notifications

**Applies to:** End users  
**Last updated:** 2026-01-11

## Overview
Configure how TaskFlow notifies you about assignments, mentions, and project updates.

## Steps
1. Go to `Settings > Personal settings`.
2. Select **Notifications**.
3. Choose which events trigger notifications (for example, assignments and mentions).
4. Choose delivery method:
   - In-app notifications
   - Email notifications
5. Select **Save changes**.

## Result
You receive notifications based on your selected preferences.
