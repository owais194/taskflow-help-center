# Create tasks

**Applies to:** End users  
**Last updated:** 2026-01-11

## Overview
Tasks track work items inside a project.

## Steps
1. Open a project.
2. Select **Add task**.
3. Enter a `Task title`.
4. (Optional) Add:
   - Description
   - Due date
   - Assignee
5. Select **Create task**.

## Result
The task appears in your project and can be assigned, prioritized, and tracked.
