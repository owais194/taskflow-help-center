# Assign tasks and use @mentions

**Applies to:** End users  
**Last updated:** 2026-01-11

## Overview
Assign tasks to teammates and use @mentions to notify someone in comments.

## Assign a task
1. Open a task.
2. Select the **Assignee** field.
3. Choose a workspace member.
4. Select **Save** (if prompted).

## Use @mentions
1. Open the task’s **Comments** section.
2. Type `@` and start typing a teammate’s name.
3. Select the correct user from the list.
4. Write your message and select **Post**.

## Result
The assignee is responsible for the task. Mentioned users receive a notification.
